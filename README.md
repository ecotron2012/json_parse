# json_parse
[![C/C++ CI](https://github.com/ecotron2012/json_parse/actions/workflows/c-cpp.yml/badge.svg)](https://github.com/ecotron2012/json_parse/actions/workflows/c-cpp.yml)

Simple JSON parser, made as a solution for John Crickett's [JSON parser Code Challenge](https://codingchallenges.fyi/challenges/challenge-json-parser) using C.

To install, run:

```bash
sudo make install
```

## Usage
```bash
json_parse file1 file2 ...
```
